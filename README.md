# c-linked-list

In its first version, this repository were just a linked list in **ANSI C** language, but with the intention of learn another programming languages, I decided to use this repository as a collection of different implementations of a linked list data structure.

> Just like the GCC compiler did renaming it from `GNU C Compiler` to `GNU Compiler Collection`, I decided to reuse this repository name as `Collection Linked List` from `C Linked List`
